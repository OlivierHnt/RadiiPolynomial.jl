Functionals

A Functional is a structure representing a compactly supported element of the dual space of some sequence space. More precisely, a Functional has an effective domain. The effective domain of a functional is defined a SequenceSpace with the property that the functional and its restriction to the effective domain coincide.

Action

The action of a Functional is performed by the left product of a Sequence with a Functional.

Defining Functional with an effective domain allows different subspaces of a given sequence space to interact:

julia> using RadiiPolynomial

julia> f = Functional(Taylor(2), [1.0, 1/2, 1/3]);

julia> a = Sequence(Taylor(3), [1.0, 2.0, 3.0, 4.0]);

julia> f*a
3.0

julia> b = Sequence(Taylor(1), [1.0, 2.0]);

julia> f*b
2.0
Note

Functional is callable: (A::Functional)(b::Sequence) = A*b.
