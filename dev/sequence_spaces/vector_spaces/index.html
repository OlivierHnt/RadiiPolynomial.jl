Vector spaces

RadiiPolynomial defines a variety of vector spaces to represent the Banach space on which one applies the Radii Polynomial Theorem.

All spaces mentioned below are a subtype of the abstract type VectorSpace.

VectorSpace
├─ CartesianSpace
│  ├─ CartesianPower
│  └─ CartesianProduct
├─ ParameterSpace
└─ SequenceSpace
   ├─ BaseSpace
   │  ├─ Chebyshev
   │  ├─ Fourier
   │  └─ Taylor
   └─ TensorSpace
RadiiPolynomial.VectorSpace — Type
VectorSpace

Abstract type for all vector spaces.

Parameter space

A ParameterSpace represents the commutative field of a parameter. This is the standard space to use for an unfolding parameter.

julia> 𝒫 = ParameterSpace()𝕂
julia> dimension(𝒫)1
julia> indices(𝒫)Base.OneTo(1)

ParameterSpace <: VectorSpace

Parameter space corresponding to a commutative field.

Example

julia> ParameterSpace()

Taylor

For a given order $n$, a Taylor sequence space is the span of $\{\phi_0, \dots, \phi_n\}$ where $\phi_k(t) := t^k$ for $k = 0, \dots, n$ and $t \in [-\nu, \nu]$ for some appropriate $\nu > 0$.

julia> 𝒯 = Taylor(1)Taylor(1)
julia> order(𝒯)1
julia> dimension(𝒯)2
julia> indices(𝒯)0:1
RadiiPolynomial.Taylor — Type
Taylor <: BaseSpace

Taylor sequence space whose elements are Taylor sequences of a prescribed order.

Examples

julia> s = Taylor(2)
Taylor(2)

julia> order(s)
2
source

Fourier

For a given order $n$ and frequency $\omega$, a Fourier sequence space is the span of $\{\phi_{-n}, \dots, \phi_n\}$ where $\phi_k(t) := e^{i \omega k t}$ for $k = -n, \dots, n$ and $t \in \mathbb{R}/2\pi\omega^{-1}\mathbb{Z}$.

julia> ℱ = Fourier(1, 1.0)Fourier{Float64}(1, 1.0)
julia> order(ℱ)1
julia> frequency(ℱ)1.0
julia> dimension(ℱ)3
julia> indices(ℱ)-1:1
RadiiPolynomial.Fourier — Type
Fourier{<:Real} <: BaseSpace

Fourier sequence space whose elements are Fourier sequences of a prescribed order and frequency.

Examples

julia> s = Fourier(2, 1.0)
Fourier{Float64}(2, 1.0)

julia> order(s)
2

julia> frequency(s)
1.0

Chebyshev

For a given order $n$, a Chebyshev sequence space is the span of $\{\phi_0, \phi_1, \dots, \phi_n\}$ where $\phi_0(t) := 1$, $\phi_1(t) := t$ and $\phi_k(t) := 2 t \phi_{k-1}(t) - \phi_{k-2}(t)$ for $k = 2, \dots, n$ and $t \in [-1, 1]$.

It is important to note that the coefficients $\{a_0, a_1, \dots, a_n\}$ associated to a Chebyshev space are normalized such that $\{a_0, 2a_1, \dots, 2a_n\}$ are the actual Chebyshev coefficients.

julia> 𝒞 = Chebyshev(1)Chebyshev(1)
julia> order(𝒞)1
julia> dimension(𝒞)2
julia> indices(𝒞)0:1
RadiiPolynomial.Chebyshev — Type

Chebyshev sequence space whose elements are Chebyshev sequences of a prescribed order.

Examples

julia> s = Chebyshev(2)

julia> order(s)

Tensor space

A TensorSpace is the tensor product of some BaseSpace. The standard constructor for TensorSpace is the ⊗ (\otimes<TAB>) operator.

julia> 𝒯_otimes_ℱ_otimes_𝒞 = Taylor(1) ⊗ Fourier(1, 1.0) ⊗ Chebyshev(1) # TensorSpace((Taylor(1), Fourier(1, 1.0), Chebyshev(1)))Taylor(1) ⊗ Fourier{Float64}(1, 1.0) ⊗ Chebyshev(1)
julia> nspaces(𝒯_otimes_ℱ_otimes_𝒞)3
julia> order(𝒯_otimes_ℱ_otimes_𝒞)(1, 1, 1)
julia> frequency(𝒯_otimes_ℱ_otimes_𝒞, 2)1.0
julia> dimension(𝒯_otimes_ℱ_otimes_𝒞)12
julia> dimensions(𝒯_otimes_ℱ_otimes_𝒞)(2, 3, 2)
julia> indices(𝒯_otimes_ℱ_otimes_𝒞)TensorIndices{Tuple{UnitRange{Int64}, UnitRange{Int64}, UnitRange{Int64}}}((0:1, -1:1, 0:1))
RadiiPolynomial.TensorSpace — Type

Cartesian power

A CartesianPower is the cartesian product of an identical VectorSpace. The standard constructor for CartesianPower is the ^ operator.

julia> 𝒯² = Taylor(1) ^ 2 # CartesianPower(Taylor(1), 2)Taylor(1)²
julia> nspaces(𝒯²)2
julia> dimension(𝒯²)4
julia> indices(𝒯²)Base.OneTo(4)
RadiiPolynomial.CartesianPower — Type
CartesianPower{<:VectorSpace} <: CartesianSpace

Cartesian space resulting from the cartesian product of the same VectorSpace.

See also: ^, × and CartesianProduct.

Examples

julia> s = CartesianPower(Taylor(1), 3)
Taylor(1)³

julia> space(s)
Taylor(1)

julia> nspaces(s)
3

Cartesian product

A CartesianProduct is the cartesian product of some VectorSpace. The standard constructor for CartesianProduct is the × (\times<TAB>) operator.

julia> 𝒫_times_𝒯 = ParameterSpace() × Taylor(1) # CartesianProduct((ParameterSpace(), Taylor(1)))𝕂 × Taylor(1)
julia> nspaces(𝒫_times_𝒯)2
julia> dimension(𝒫_times_𝒯)3
julia> indices(𝒫_times_𝒯)Base.OneTo(3)
RadiiPolynomial.CartesianProduct — Type
CartesianProduct{<:Tuple{Vararg{VectorSpace}}} <: CartesianSpace
