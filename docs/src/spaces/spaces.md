# Spaces

The spaces under consideration are the ones where the elements are compactly supported sequences.

The abstract type [`SequenceSpace`](@ref) has the two subtypes [`UnivariateSpace`](@ref) and [`TensorSpace`](@ref) where the latter corresponds to the tensor products of some spaces in [`UnivariateSpace`](@ref).
